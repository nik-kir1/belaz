# ASCII Art Converter

This repository contains a Python script (`ascii_art.py`) that converts images to ASCII art. The script uses an extended set of characters to provide more detailed output.

## Requirements

- Python 3
- Pillow (`pip install pillow`)

## Usage

```bash
python ascii_art.py path/to/image.png --width 100
```

The width argument controls the number of characters per line in the generated ASCII art.
